#### 1. 파이썬은 인간다운 언어이다

```
if 4 in [1, 2, 3, 4]: print("4가 있습니다")
```
이 소스는 다음처럼 직곽적인 해석이 가능하다
```
만약 4가 1, 2, 3, 4 중에 있으면 "4가 있습니다"를 출력한다.
```

파이썬으로 할 수 있는 일
---------

##### 1. 웹 프로그래밍
##### 2. 인공지능과 머신러닝
대표적으로 사이킷런(scikit-learn), 텐서플로(TensorFlow), 파이토치(PyTorch), 케라스(Keras) 등이 있다.
##### 3. 수치 연산 프로그래밍
파이썬 자페는 C 같은 언어보다 수치 연산 속도가 느리다.
하지만 NumPy 라는 수치 연산 모듈은 내부적으로 C로 작성되어 빠른 속도를 제공하면서도, 파이썬의 간결한 문법으로 사용할 수 있어 이 문제를 해결할 수 있다.
##### 4. 데이터 분석
NumPy, Pandas, Matplotlib 등의 라이브러리로 데이터 처리, 통계 분석, 시각화를 손쉽게 할 수 있다.
##### 5. 데이터베이스 프로그래밍
파이썬은 Sybase, Informix, Oracle, MySQL, PostgreSQL 등의 데이터베이스에 접근하기 위한 도구를 제공한다.

pickle이라는 모듈을 사용해 파이썬 자료를 변형없이 파일에 저장하거나 불러올 수 있다.
##### 6. 시스템 유틸리티 제작
파이썬은 운영체제의 시스템 명령어를 사용할 수 있는 각종 도구를 갖추고 있으며 이를 바탕으로 파일 관리, 로그 분석, 시스템 모니터링 등 여러가지 시스템 유틸리티를 만들 수 있다.
##### 7. GUI 프로그래밍
Tkinter를 사용하면 단 5줄의 코드만으로 윈도우 창을 띄울 수 있다.
##### 8. C/C++ 와 결합
C나 C++로 만든 프로그램을 파이썬에서 사용할 수 있고, 파이썬으로 만든 프로그램도 C나 C++에서 사용할 수 있다.
